# Olist E-Commerce Power BI Dashboard

Welcome to our Power BI visualization project! In this project, I analyzed and visualized the sales performance of the Brazilian e-commerce dataset *Olist*.

##  Project Objective
This dashboard transforms raw e-commerce data into a clear, interactive visual summary. With a simple design, it helps users quickly understand order volumes and delivery efficiency.

## 💡 Key Findings and Visualizations
* *Top KPIs:* Total revenue (431 million) and total order volume (110,000+) were tracked.

* *Revenue by City:* São Paulo's dominance on overall sales was highlighted using strategic color contrast.

* *Sales by Category:* Top-performing product categories were visualized understand customer demand.

* *Logistics Performance:* Average delivery time (12.41 days) was monitored to track operational efficiency.

##  Tools Used
* *Power BI:* Data visualization, KPI card creation, and dashboard user interface design.
